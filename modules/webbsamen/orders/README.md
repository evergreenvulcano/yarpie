# Webbsamen Content Orders

Repo orders live here because product-level text needs are discovered in the web surface.

This is not the writing desk. Orders are sent to:

```text
LOMPOLO-CORE/_redaktionell_contentlogik/03_artifacts/kontent_skribent/
```

Krönikörsamen belongs inside `kontent_skribent`, not as a separate repo module.

## Rule

Webbsamen does not write free content when a page lacks body. He orders exact text needs:

- krönikerande brödtext,
- eventkort,
- signalnotis,
- mottagarkort,
- protokollkant,
- 404-/stängd-dörr-copy,
- mikrocopy,
- övergångstext,
- läsnästa-copy,
- human-review-fråga.

Returned text remains candidate material until sluss/review allows it back into HTML.
